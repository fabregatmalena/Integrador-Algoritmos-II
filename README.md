# Juego del Ahorcado - Proyecto Integrador

Este proyecto fue desarrollado como parte del Proyecto Integrador para la materia "Algoritmos y Estructuras de Datos II".
Es una implementación en lenguaje C del clásico juego del "Ahorcado". El programa selecciona palabras de un archivo de texto y el jugador debe adivinarlas letra por letra antes de que se completen los 6 intentos fallidos.


# Guía de Ejecución: Juego del Ahorcado (Dev-C++)

Antes de comenzar, asegúrate de tener los siguientes 4 archivos guardados **juntos en la misma carpeta**:

* `ahorcado_final.c` (Código fuente principal).
* `cola.h` (Librería para el manejo de colas).
* `pila.h` (Librería para el manejo de pilas).
* `palabras.txt` (Archivo de datos necesario para el juego).

## 2. Abrir el Proyecto

1.  Inicia **Dev-C++**.
2.  Dirígete al menú **Archivo > Abrir** (File > Open).
3.  Navega hasta tu carpeta y selecciona únicamente el archivo **`ahorcado.c`**.

## 3. Compilar y Ejecutar

Para iniciar el juego, simplemente:

* Presiona la tecla **`F11`** en tu teclado.
* O bien, selecciona la opción **Ejecutar > Compilar y Ejecutar** en el menú superior.

Dev-C++ generará el ejecutable automáticamente y abrirá la consola con el juego listo para usar.
